Designul responsive se bazează pe grid-uri fluide, imagini flexibile și media queries. Utilizăm o abordare mobile-first, asigurându-ne că experiența pe dispozitive mobile este la fel de bună ca pe desktop. Framework-uri precum Tailwind CSS sau Bootstrap accelerează dezvoltarea.

UI (User Interface) se referă la aspectul vizual: culori, tipografie, butoane, spațiere. UX (User Experience) vizează întreaga experiență a utilizatorului: arhitectura informației, fluxurile de navigare, accesibilitatea și ușurința în utilizare. Ambele sunt esențiale pentru un produs de succes.

🎨 UI este puntea, UX este întreaga călătorie

Flexbox este ideal pentru layout-uri unidimensionale (rând sau coloană) și distribuirea spațiului între elemente. CSS Grid excelează la layout-uri bidimensionale (rânduri și coloane simultan). Cel mai adesea, le combinăm: Grid pentru structura macro a paginii, Flexbox pentru componentele interne.

📐 Grid pentru structură 🔄 Flexbox pentru componente

Variabilele CSS permit definirea unor valori reutilizabile (culori, spațieri, tipografie) și actualizarea lor dinamică. Sunt utile pentru teme întunecate/deschise, întreținere mai ușoară și consistență vizuală. Pot fi modificate în timp real cu JavaScript fără a rescrie stiluri.

🎨 :root { --primary: #2c7f7f; } · suport în toate browserele moderne

Accesibilitatea implică: contrast suficient între text și fundal, navigare completă cu tastatura, atribute ARIA corecte, text alternativ pentru imagini și structură semantică HTML (header, main, nav, etc). Testează cu axe sau Lighthouse pentru conformitate WCAG 2.1 nivel AA.

♿ aria-* attributes ⌨️ keyboard navigation

Web font-urile (Google Fonts, Adobe Fonts) îmbunătățesc designul, dar pot încetini încărcarea. Recomandări: preload pentru fonturi critice, subsetare (doar caracterele necesare), font-display: swap pentru a evita FOIT (flash of invisible text) și utilizarea a maxim 2-3 greutăți (weights).
